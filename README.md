# Backend-
Telegram-bot Tinder(Иметация общения со звёздами используя ВСТРОЕННЫЙ ChatGPT, 
Генерация профайла используя 5 вопросов для приглашения на свидание)

Telegram-bot DotaProTracker(Альтернатива сайту https://dota2protracker.net, позволяет узнать какие персонажи нынче популярны и что на них собирать с помощью парсинга сайта и API OpenDota,
так же много фишек в разработке)

valentinka(Программа сделана на основе библиотеки - pygame, создаёт окно с прглашением стать валентином, при нажатие на кнопку 'Нет' она постеменно уменьшается и перемещаеться в рандомное место на экране пока не исчезнет)

CatLocker(Программа работает как известный вирус WinLocker. Код открывает незакрываемое окно из которое нельзя свернуть или выйти в другое окно и загружает себя в папку автозагрузуи, так что после перезахода он опять включиться)
